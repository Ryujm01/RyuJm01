## About Me

안녕하세요, `SkyMin`입니다.  
**사용자 화면부터 API, 데이터 흐름, 서버 환경까지 직접 연결해서 만드는 개발자**입니다.

- 단순 기능 구현이 아니라 **서비스 단위로 설계하고 끝까지 연결하는 것**을 지향합니다.
- `React`, `Next.js`, `Flutter`로 UI를 만들고  
  `FastAPI`, `Docker`, `Nginx`, `DB`까지 직접 구성해봤습니다.
- 공개 저장소보다 `private repository` 기반 실무/개발 경험 비중이 높습니다.

---

## What I Build

- **Web Products** : 랜딩 페이지, 상세 페이지, 계산 로직, 폼, 반응형 UI
- **Backend / Infra** : API 서버, reverse proxy, 로컬 환경, DB 연결, 트러블슈팅
- **Mobile App** : Flutter 기반 UI, 이미지 업로드, 디바이스 대응
- **Automation** : Google Sheets 기반 데이터 처리 및 성능 개선
- **AI / Vision** : YOLO + OpenCV 기반 영상 처리 파이프라인

## Proof Of Work

- `Next.js` 기반 자동차 금융 웹서비스에서 **랜딩 + 상세 페이지 + 카드 혜택 계산 로직**을 직접 구현

- `Flutter` 앱에서 **이미지 업로드 + 미리보기 + overflow UI 문제 해결**
- `YOLO + OpenCV + Streamlit` 기반 **영상 객체 탐지 및 프레임 처리 파이프라인 구성**
