## About Me

안녕하세요, `SkyMin`입니다.  
`React`와 `JavaScript`를 중심으로 사용자에게 편하고 보기 좋은 웹 서비스를 만드는 걸 좋아합니다.

- 다양한 기능이 있는 서비스를 화면 단위로 정리하고 연결하는 작업에 흥미가 있습니다.
- `CSS`, `MUI`, `styled-components`로 UI를 다듬고 인터랙션을 더하는 걸 즐깁니다.
- 커뮤니티, 관리, 채팅, 쇼핑처럼 여러 기능이 함께 있는 프론트엔드 구조를 꾸준히 경험하고 있습니다.

## Featured Project

### FITME

여러 기능을 하나의 서비스 흐름으로 연결한 `React` 기반 프론트엔드 프로젝트입니다.

- `운동 / 식단 관리`, `커뮤니티`, `채팅`, `쇼핑`, `관리자 페이지`까지 포함된 구조를 경험했습니다.
- `React Router`로 다양한 페이지를 연결하고, `MUI`와 `styled-components`로 화면을 구성했습니다.
- `SockJS`, `STOMP`, `socket.io-client` 같은 라이브러리를 사용해 실시간 기능도 다뤘습니다.
