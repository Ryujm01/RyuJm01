## About Me

안녕하세요, `SkyMin`입니다.  
웹, 모바일, 데이터베이스, 자동화, 비전 AI까지 폭넓게 다루며 실제로 동작하는 서비스를 만드는 개발자입니다.

- `Python`, `TypeScript`, `JavaScript`, `C#`, `Dart`를 사용해 기능을 설계하고 구현해왔습니다.
- `React`, `Next.js`, `Flutter` 기반의 사용자 화면부터 `FastAPI`, `SQL`, `Docker`, `Nginx` 기반 개발 환경까지 경험했습니다.
- 공개 저장소보다 `private repository`에서 진행한 작업 비중이 높아, 이 README는 실제 경험 중심으로 정리했습니다.

## Focus

- `Web Frontend` : 랜딩 페이지, 상세 페이지, 폼, 계산 로직, 반응형 UI
- `Backend / Infra` : FastAPI, Docker Compose, Nginx, PostgreSQL, MySQL 환경 구성과 트러블슈팅
- `Mobile App` : Flutter 기반 UI 구현, 이미지 업로드, 디바이스별 레이아웃 보정
- `Automation` : Google Apps Script와 Sheets 기반 데이터 처리 및 병목 개선
- `AI / Vision` : YOLO, OpenCV, Streamlit 기반 객체 탐지 파이프라인 구성
- `Game / Logic` : Unity와 C#으로 인벤토리, 자원 채집, 제작 흐름 구현

## Selected Work

### Web / Frontend

- `AutoFriends` : 자동차 금융·상담형 웹서비스 프론트엔드를 `Next.js` 기반으로 설계하고, 랜딩 페이지, 상세 페이지, 카드 혜택 계산 로직, 폼 UI, slug 라우팅, 반응형 화면까지 구현했습니다.
- `Legacy PHP Site Analysis` : 기존 `PHP` 운영 사이트의 include 구조, 공통 설정, SQL 조회 흐름, DB 연결 방식을 분석해 `Next.js` 전환 가능 구조를 정리했습니다.
- `FITME` : `React` 기반으로 운동, 식단, 커뮤니티, 채팅, 쇼핑, 관리자 페이지가 연결된 다기능 웹 프로젝트를 다뤘습니다.

### Backend / Infra / Ops

- `FastAPI Local Stack` : `FastAPI`, `Docker Compose`, `Nginx`, `MySQL` 조합으로 로컬 백엔드 환경을 구성하고 `bad gateway`, 포트 매핑, 프록시 연결 문제를 점검했습니다.
- `PostgreSQL Local Setup` : macOS 환경에서 `PostgreSQL` 설치, 프로세스 확인, `psql` 접속, 계정과 DB 연결을 테스트하며 개발용 DB 환경을 구성했습니다.
- `Admin Flow Testing` : 관리자 계정 로그인, 권한 부여, 사용자 관리, CRUD, 실시간 반영, 푸시 기능 검증 등 운영 기능 테스트를 수행했습니다.

### Automation

- `Apps Script Monitoring Tool` : `Google Apps Script`와 `Google Sheets` 기반 업무 도구에서 조회 병목을 분석하고 `getValue`, `getValues`, `setValue` 사용 구조를 정리해 로딩 성능 개선을 시도했습니다.

### Mobile / AI / Game

- `Flutter UI Fixes` : `Flutter` 앱에서 이미지 업로드 및 미리보기 UI를 구현하고 `bottom overflowed` 문제를 해결해 기기별로 유연한 화면이 되도록 조정했습니다.
- `YOLO Video Detection` : `Ultralytics YOLO`, `OpenCV`, `Streamlit`을 활용해 영상 프레임 단위 객체 탐지, 고신뢰 프레임 추출, 결과 저장, 후속 GPS 연동 구조를 설계했습니다.
- `Unity Survival Systems` : `Unity`와 `C#`으로 인벤토리, 자원 채집, 아이템 획득 등 생존형 게임의 핵심 시스템을 구현했습니다.
