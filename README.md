- `Unity`와 `C#`으로 인벤토리, 자원 채집, 아이템 획득 같은 게임 시스템도 구현했습니다.

## Selected Work

### Web / Frontend

- `AutoFriends` : 자동차 금융·상담형 웹서비스 프론트엔드를 `Next.js` 기반으로 설계하고, 랜딩 페이지, 상세 페이지, 카드 혜택 계산 로직, 폼 UI, slug 라우팅, 반응형 UI를 구현했습니다.
- `Legacy PHP Site Analysis` : 기존 `PHP` 운영 사이트의 include 구조, 공통 설정, SQL 조회 흐름, DB 연결 구조를 분석하고 신규 웹 전환 가능 구조를 검토했습니다.
- `FITME` : 운동, 식단, 커뮤니티, 채팅, 쇼핑, 관리자 페이지가 연결된 `React` 기반 웹 프로젝트를 다뤘습니다.

### Backend / Infra / Ops

- `FastAPI Local Stack` : `FastAPI`, `Docker Compose`, `Nginx`, `MySQL` 조합으로 로컬 백엔드 환경을 구성하고 `bad gateway`, 프록시, 포트 연결 문제를 점검했습니다.
- `PostgreSQL Local Setup` : macOS 환경에서 `PostgreSQL` 설치, 프로세스 확인, `psql` 접속, 계정과 DB 연결을 테스트하며 개발용 DB 환경을 구성했습니다.
- `Admin Flow Testing` : 관리자 권한 로그인, 사용자 관리, CRUD, 실시간 반영, 푸시 기능 검증 등 운영 플로우 테스트를 수행했습니다.

### Automation / Mobile / AI / Game

- `Apps Script Monitoring Tool` : `Google Apps Script`와 `Google Sheets` 기반 내부 도구에서 조회 병목을 분석하고 데이터 로딩 구조를 개선했습니다.
- `Flutter UI Fixes` : `Flutter` 앱에서 이미지 업로드 및 미리보기 UI를 구현하고 overflow 문제를 해결해 기기별 대응형 화면으로 보정했습니다.
- `YOLO Video Detection` : `Ultralytics YOLO`, `OpenCV`, `Streamlit`을 활용해 영상 프레임 객체 탐지, 결과 저장, 후속 GPS 연동 구조를 설계했습니다.
- `Unity Survival Systems` : `Unity`와 `C#`으로 인벤토리, 자원 채집, 아이템 획득 등 생존형 게임 시스템을 구현했습니다.
